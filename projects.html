PROJECTS

Either personal projects I made or team projects I collaborated on

Game Info Searcher

Created a JavaFX application allowing users to search for PC games by certain categories/filters which displays information regarding the game's title, price, and discount price (if there is a sale). Available categories include:

  • Best/worst deals
  • Lowest/highest price
  • Highest/lowest rating

The program also lets users to convert the prices to various global currencies (such as USD, EUR, CNY, etc). The application utilizes two external APIs: CheapShark API and FXRates API.

Chase the Fun

A 2D top-down space game created by a team of programmers, artists, and musicians. I mainly contributed as a programmer by implementing the player and enemy movement. I also designed most of the user interface such as the bottom "modules" and the pause menu.
